# Intallation:

## Environment Variables

set the following environment variable with the url enpoint from the API backend e.g:

```
REACT_APP_API_URL = http://localhost:4000/api
```

## Run backend

```
$ start
npm install
npm run start
```

mensaje de ok:

```
Servidor corriendo en puerto 4000
DB online!
```
### Run frontend

```
$ start
npm install
npm run start
```
